Site Update February 2023

published on 23 Feb 2023

It’s been a while since I last updated the site so here’s a rundown on what’s new.

New Content

There is now a library page where you can see all the books I own. It is an ongoing project and there are a lot of books missing from it that I have yet to catalogue. However if you find anything of interest on it and would like to contact me, please do so.

You can now find the long awaited Cookbook project at the recipes page. It is a long awaited feature I am sure. It’s a bit empty at the moment but I will be filling it out from time to time. Read the Cookbook Now Live article linked below to find out more.

The domain name has now changed from dunkelheit.biz to dunkelheit.xyz and the old domain will expire within a month.

New Articles:

  • Having Fun Is Trad
  • Against CDNs
  • Necessary Evils on the Internet
  • Every Day Carry and Equipment
  • Cookbook Now Live

Housekeeping

The home page and about me pages have been updated to include more accurate information and restructured a bit to ensure clarity. Please keep in mind that the YouTube and Odysee links are placeholders while I set those up. I’ll update this post when it has been fixed.

Known Issues

Email is down, I will attempt to fix it at some point when I am less busy. For now you can contact me via twitter @Dunkelheitxyz. If you are familiar with my content you will know that I am against social media and twitter in particular is my usual go to example of why. Check the Necessary Evils on the Internet article linked above to see some of my more recent thinking on the matter. I am still not convinced it is worth keeping a twitter account long term, for now it is a stopgap fix for a problem I don’t have the time to find a better solution to.

The YouTube and Odysee links don’t work. This is on purpose, they are placeholders.
